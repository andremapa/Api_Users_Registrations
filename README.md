# Api de Cadstro de Usuários

## Sobre o projeto

  Projeto desenvolvido para cadastrar usuários por meio de uma Api hospedada localmente
  na porta 8080.

## Tecnologias usadas:
  - Java 17
  - Sprinboot 2.7.0
  - PostgreSQL
  
## Principais dependencias do ecossistema Spring:
  - Spring Web
  - Spring Data Jpa
  - Spring Hateaos
  - PostgreSQL Driver
  - Validation
